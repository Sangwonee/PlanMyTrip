# PlanMyTrip

AI가 여행 일정을 자동으로 짜주는 챗봇 서비스입니다.  
사용자가 지역, 날짜, 여행 스타일 등을 입력하면, 한국관광공사 TourAPI에서 실제 장소를 가져오고 OpenAI GPT가 최적의 일정을 생성해줍니다.

---

## 주요 기능

- 💬 **AI 챗봇 인터페이스** — 대화 형식으로 여행 정보(지역, 날짜, 교통, 여행 스타일)를 입력
- 🏖️ **실제 장소 기반 일정 생성** — 한국관광공사 TourAPI로 실제 관광지·음식점·숙소 데이터를 조회
- 🤖 **플래너** — 후보 장소 중 최적 조합으로 날짜별 일정 자동 생성
- 🗺️ **카카오맵 연동** — 생성된 일정의 장소를 지도 위에 마커·경로로 시각화
- 🗂️ **사이드바 채팅 히스토리** — 이전 대화 목록 관리
